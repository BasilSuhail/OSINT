**Goal:** Make `/brain/ask` answer like a real analyst: 4b model per-ask, uncertainty/contested framing in the prompt, an honest no-evidence fallback, and visibly-draft streaming.

**Architecture:** New `qa_model`/`qa_min_free_mb` settings wired through both ask endpoints + the citation repair call (`keep_alive="0"` so 4b never lingers); prompt + fallback changes in `app/brain/qa.py`; `qa_rubric` treats the new no-evidence answer as a refusal; frontend `AskBox` renders a draft state until the verified final. Spec: `docs/superpowers/specs/2026-07-15-qa-v2-design.md`.

**Tech Stack:** Python 3.14, FastAPI, pytest, ruff; Next.js/TS, vitest, eslint.

```python
def build_cited_fallback_answer(
    stories: list[dict[str, Any]], question: str | None = None
) -> str:
    """Deterministic answer when the model cannot repair missing citations."""
    if not stories:
        return REFUSAL_ANSWER
    story = stories[0]
    title = str(story.get("title") or "").strip()
    if not title:
        return REFUSAL_ANSWER
    if question is not None:
        terms = _question_terms(question)
        text = f"{title} {story.get('gist') or ''}".lower()
        if terms and not any(term in text for term in terms):
            return NO_EVIDENCE_ANSWER
```
